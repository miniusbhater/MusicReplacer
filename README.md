# MusicReplacer
Replaces the default in game music for SimplePlanes 2
<br /><img width="294" height="396" alt="image" src="https://github.com/user-attachments/assets/5c7c89a0-8f40-4fe2-911a-68e6aa527980" />
<br />Press Y to toggle the GUI
<br />Supports .wav and .mp3 audio files
<br />Plays Songs in a random order
<br />You can select songs to play from the GUI
<br />[SP2 Notification Library](https://github.com/miniusbhater/SP2-Notification-Library) Strongly recommended
<br />Remove the README file from the Music folder before first launch
